Репозиторий для проекта на зачёт

Перед запуском требуется в Docker установить PostgreSQL с помощью команды 

```
docker compose up -d
```

После запуска контейнера можно запускать приложение, 
в коде при использовании Intellij Idea через иконку Play в файле TaskTrackerApplication

Пользователь АДМИН уже добавлен в базу данных после запуска приложения

Его mail и пароль - admin
Можно через sign in зайти под ним и начать с админовских эндпойнтов
Потом нужно создать пользователя sign up
Зайти под ним

И сперва создать группу задач на него, после уже создавать задачи
